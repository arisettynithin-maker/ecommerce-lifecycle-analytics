# E-commerce Lifecycle Analytics

End-to-end analytics project analyzing **~2 million e-commerce user events** to understand **customer behaviour, conversion funnels, retention patterns, and user segmentation**.

The project combines **SQL, Python, and Power BI** to transform raw event data into actionable insights for product and marketing teams.

---

# Project Overview

This project analyzes **1,999,840 e-commerce events generated by 935,717 users** interacting with an online store.

Events include:

- Product views  
- Cart additions  
- Purchases  

The goal is to understand:

- Customer conversion behaviour  
- Funnel drop-off points  
- Retention patterns  
- High-value customer segments  

---

# Key Funnel Metrics

The project reconstructs the full **e-commerce conversion funnel**.

| Funnel Stage | Events |
|------|------|
| Product Views | **586,694** |
| Add to Cart | **20,259** |
| Purchases | **16,538** |

Conversion rates:

| Conversion Step | Rate |
|------|------|
| View → Cart | **3.45%** |
| Cart → Purchase | **81.63%** |
| View → Purchase | **2.81%** |

Insights:

- Most users **drop off between View and Cart**, indicating product page friction.
- Once a user adds an item to the cart, **purchase likelihood is very high**.

---

# User Engagement Metrics

User interaction analysis shows how frequently users return and engage with the platform.

| Metric | Value |
|------|------|
| Avg sessions per user | **1.68** |
| Same-day repeat users | **29.97%** |
| 7-day return rate | **16.28%** |

Insights:

- Most users interact **only once**, suggesting low early retention.
- Engagement decreases sharply after the first session.

# Key Business Insights

1. **Cart behaviour is highly predictive of purchase**  
   Users who add items to cart convert at **81.6%**.

2. **Major drop-off occurs early in the funnel**  
   Only **3.45% of product views** lead to cart additions.

3. **Customer retention is low after the first interaction**  
   Only **16.28% of users return within 7 days**.

4. **High-value segments represent a small portion of users but drive disproportionate revenue**.

---

# Business Recommendations

Based on the analysis:

1. Improve **product page conversion** through better product information and UX optimization.
2. Introduce **cart reminders and retargeting campaigns** to recover abandoned carts.
3. Develop **loyalty or retention programs** to improve repeat engagement.
4. Personalize recommendations for **high-value customers**.
