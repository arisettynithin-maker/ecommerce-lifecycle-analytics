# Data Dictionary

This document describes the key variables used in the **E-commerce Lifecycle Analytics** project.

The dataset contains **~1,999,840 user interaction events** generated by **~935,717 users** during October 2019.

Each row represents a **single user interaction event** on the e-commerce platform.

---

# Core Dataset Fields

| Column        | Description                                     |
| ------------- | ----------------------------------------------- |
| event_time    | Timestamp when the user interaction occurred    |
| event_type    | Type of user interaction (view, cart, purchase) |
| product_id    | Unique identifier of the product                |
| category_id   | Unique identifier of the product category       |
| category_code | Hierarchical product category                   |
| brand         | Product brand name                              |
| price         | Product price in USD                            |
| user_id       | Unique identifier for each user                 |
| user_session  | Unique identifier for a browsing session        |

---

# Derived Fields Created During Analysis

Additional features were created during the **data preparation and feature engineering stage**.

| Column        | Description                                       |
| ------------- | ------------------------------------------------- |
| main_category | Top-level category extracted from `category_code` |
| sub_category  | Subcategory extracted from `category_code`        |
| price_band    | Price category assigned based on product price    |
| event_date    | Date extracted from `event_time`                  |

---

# Event Type Definitions

| Event Type | Description                           |
| ---------- | ------------------------------------- |
| view       | User viewed a product page            |
| cart       | User added a product to their cart    |
| purchase   | User completed a purchase transaction |

These event types form the basis of the **conversion funnel analysis**:

```
View → Cart → Purchase
```

---

# Analytical Tables Generated

Several aggregated datasets were created during the analysis:

| Dataset                      | Description                                |
| ---------------------------- | ------------------------------------------ |
| user_summary                 | User-level behavioral metrics              |
| user_funnel                  | User conversion funnel metrics             |
| price_band_funnel_summary    | Conversion performance by price range      |
| category_funnel_summary      | Conversion performance by product category |
| customer_segmentation_output | Customer behavioral segments               |

These processed datasets are stored in:

```
data/processed/
```
